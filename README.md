# 任意视频倍速播放

功能如下：

① +/-0.25倍速：按w/s键；

② 切换整数倍速：按1/2/3/4键；

③ +/-0.5s：按→/←键；

④ +/-10%音量：按↑/↓键。

注：目前测试了mooc和本地视频，都可以，如果有不可以的网站请反馈给我。

注：用浏览器播放本地视频，需要设置油猴插件“允许访问文件URL”。
